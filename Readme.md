# Student Productivity Dashboard

A modern and responsive productivity dashboard designed to help students organize tasks, deadlines, priorities, and academic projects efficiently.

## Overview

Student Productivity Dashboard is a front-end web application built using modern web technologies. It provides task management, analytics, Kanban boards, and graph-based task visualization within a single interface.

The project demonstrates software development principles, UI/UX design, data visualization, and interactive front-end development.

## Features

### Task Management

* Add, edit, and delete tasks
* Task completion tracking
* Task categorization
* Priority management
* Due date assignment
* Search and filtering

### Dashboard Analytics

* Total tasks overview
* Completed tasks tracking
* Pending tasks tracking
* High-priority task monitoring
* Interactive charts using Chart.js

### Kanban Board

* Drag-and-drop task management
* SortableJS integration
* Move tasks between workflow stages

### Graph Visualization

* Interactive task relationship graph
* Built using Cytoscape.js
* Category-to-task relationship mapping

### Modern User Interface

* Responsive design
* Bootstrap-based layout
* Interactive notifications using Toastify
* Font Awesome icons
* Mobile-friendly experience

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Shushruth18/student-task-manager.git
```

2. Open the project folder

```bash
cd student-task-manager
```

3. Open:

```text
index.html
```

in your browser.

No installation or backend setup is required.
